# 慢一点，刚好遇见你 · Slow Down, Just in Time to Meet You

一款用 **Three.js** 制作的 3D 恋爱 RPG，全年龄向。故事发生在温哥华：理科生 Allen 与花艺师塔米，在上帝 Joshua 的注视下，从一只青瓷杯开始的爱情故事。

**单文件游戏**：全部代码都在 `index.html`，无构建步骤，Three.js 从 CDN 加载。

## 🎮 两种模式

### 📖 故事模式（14 章完整剧情）
| 章节 | 地点 | 玩法 |
|---|---|---|
| 1 | 茶空间 | 初遇 · 对话选择 |
| 2 | 英吉利湾 | 双人钓鱼 🎣 + 贝壳收集 |
| 3 | 星空草甸 | 双人抓萤火虫 ✨ |
| 4 | 煤气镇 | 蒸汽钟前拍立得 📸 |
| 5 | 格兰维尔岛 | 双人分头采购 🛒 + 街头共舞 |
| 6 | 斯坦利公园 | 双人海堤骑行 🚲 + 告白 |
| 7 | 雨夜街道 | 冷战 · 一杯热奶茶 |
| 8 | 塔米的公寓 | 塔米视角 · 撸猫 🐱 · 开门的抉择 |
| 9 | 卡皮拉诺吊桥 | 大风过桥 🌉 |
| 10 | 松鸡山 | 堆雪人 ⛄ + 打雪仗 |
| 11 | 慢一点花店 | 塔米视角 · 开业前夜 💐 |
| 12 | 英吉利湾 | 烛光求婚 💍 + 烟花 |
| 13 | 女皇公园 | 婚礼 · 双人誓言 💒 |
| 14 | 家 | 几年以后 · 终章 |

- 剧情选项有**情商考验**：低情商选择会让塔米生气，心动值下降 💢
- 心动值决定结局：完美 / 温柔 / 慢慢来 三种结局 + 全程数据统计

### 🚗 自由模式（GTA 式开放世界）
- 迷你温哥华地图：十字主干道 + 海滨路，北面松鸡山雪顶、狮门大桥、科学世界
- **可驾驶汽车**（W 加速 / S 刹车倒车 / AD 转向），塔米可搭车
- 7 大故事地标可参观互动，12 颗爱心散落收集
- 右下角**小地图**实时显示位置、地标与爱心
- NPC 车流与行人

## 🕹 操作

| | P1 · Allen | P2 · 塔米 |
|---|---|---|
| 移动 | WASD | 方向键 |
| 互动 | E / 空格 | Enter |
| 对话推进 | 空格 / 点击对话框 | Enter / 点击对话框 |

单人游玩：单人视角章节两套按键通用。

## 🚀 本地运行

直接双击 `index.html` 用浏览器打开即可（需联网加载 Three.js CDN）。

## ☁️ 部署

静态单页，任何静态托管都可以：

**Vercel**：导入本仓库 → Framework 选 `Other` → 直接 Deploy（无需构建命令）。

## 🛠 技术

- Three.js r160（ES Module + Import Map，无打包）
- 程序化低多边形建模：骨骼化角色（走路摆臂 / 眨眼 / 坐姿 / 骑车 / 过场表演）
- WebAudio 合成音效（零素材依赖）
- 打字机对话系统 + 分支选项 + 表情气泡
